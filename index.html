Project Overview

We are building a secure shopping cart system for the Secure Retail System — Capstone A, Group 5. Our system protects customer data, saves cart progress, and ensures a smooth shopping experience.

🔒 Secure Login

Only authorised users can access their private shopping cart.

🛒 Cart Privacy

Each cart is locked to that user — no one else can view or edit it.

💾 Saved Progress

Your cart is saved even if you lose internet or close the app.

✅ Input Validation

Blocks invalid or fake item entries to keep the system reliable.
