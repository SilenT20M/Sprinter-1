Who We Are

Project Description

The Secure Retail System is a shopping cart module built for Capstone A, Group 5. Our system protects customer data, saves cart progress across sessions, and ensures each customer's cart remains completely private and secure.

Why We Built This

Problems We Solve

🛒 Lost Shopping Carts

Customers spend time picking items, lose internet or close the app — their cart disappears. This kills sales momentum and causes purchase abandonment. Our system saves carts on the server so they never disappear.

🔓 Weak Security

Basic online stores leave session tokens unprotected. Malicious users can see or tamper with another customer's private cart data. We lock every cart strictly to its owner.

📋 No Task Tracking

Without proper workflow tools, teams overwrite each other's work and lose code. We use GitHub Issues, branches, and Pull Requests to keep everything organised and safe.
