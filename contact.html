About Our Project

Learn about the Secure Retail System and our team

Project Description

The Secure Retail System is a shopping cart module built for Capstone A. It protects customer data, saves cart progress across sessions, and ensures each customer's cart remains completely private.

Problem Statement

Many online stores lose customer carts when internet is lost or the app is closed. Our system solves this by saving carts securely on the server. We also protect session tokens so no malicious user can access another customer's private cart data.
